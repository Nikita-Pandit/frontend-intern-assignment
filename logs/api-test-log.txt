
# API Test Logs for Frontend Intern Assignment

## 1. User Registration
POST /api/auth/register
Request:
{
  "name": "Nikita",
  "email": "nikita@example.com",
  "password": "123456"
}
Response:

{
    "token": <your token>, 
    "user": {
        "name": "Nikita",
        "email": "nikita@example.com"
    },
    "message": "User registered successfully"
}
